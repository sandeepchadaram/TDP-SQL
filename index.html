SQL Documentation

SQL Introduction

SQL is a standard language for accessing and manipulating databases.

  • SQL stands for Structured Query Language
  • SQL lets you access and manipulate databases
  • SQL can execute queries against a database
  • Although SQL is an ANSI/ISO standard, there are different versions of the SQL language. However, to be compliant with the ANSI standard, they all support at least the major commands (such as SELECT, UPDATE, DELETE, INSERT, WHERE) in a similar manner
  • SQL can set permissions on tables, procedures, and views

Connect to Database

Webpages connect to SQL databases using PHP. This is a very important step as it will allow you to input, modify or fetch data to/from the database

Creating a database involves installing MYSQL (or similar) server to host your data and then connecting your data to the webpage using the appropriate PHP scripts. Bear in mind that security of the database is very important and hence access should be restricted wherever necessary.

Creating a Database

In this example, we are using Microsoft's MYSQL software. Download the latest software and install it.

After installing the software, open the MySQL workbench. Create a new port and give the necessary login details. Store these credentials somewhere safe.

Next, MySQL will prompt you to create a new database. Give the database a name and type in the following command in the SQL editor to start using your database.

use database your_database_name;

Insert your database name at your_database_name

Querying from Database

The SELECT command is used to select data and get it from the database.

SELECT * from your_database_name;

This would show the entire data in your database. * means everything.

SELECT DISTINCT is a variant of SELECT statement. This fetches only the distinct values in the database.

SELECT DISTINCT * from you_database_name;

Next is the WHERE clause. This is used to fetch data which fulfill one or more criteria. State the condition after the WHERE clause.

SELECT * from your_database_name WHERE condition;

The INSERT INTO statement is used to insert new records in a table.

INSERT INTO table_name (column1, column2, column3, ...) VALUES (value1, value2, value3, ...);

The UPDATE statement is used to modify the existing records in a table.

UPDATE table_name SET column1 = value1, column2 = value2, ... WHERE condition;

Conclusion

SQL is extremely useful to efficiently use store, modify and use your data thereby giving an edge in drawing out meaningful insights to further your business.
